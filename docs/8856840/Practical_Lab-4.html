def model_estimation(model, y_train, y_test, y_pred_train, y_pred_test):
    """
    Evaluate the model using MAE. 
    Inputs:
        model: the model to be evaluated
        y_train: the true y values of the train dataset
        y_test: the true y values of the test dataset
        y_pred_train: the predicted y values of the train dataset
        y_pred_test: the predicted y values of the test dataset
    Returns:
        None. 
        Print the MAE of the train and test datasets and the equation of the model.
        Plots of the residuals, percentage error, and histogram of the percentage error.
    """
    # create three subplots in one row
    fig, (ax1, ax2, ax3) = plt.subplots(1, 3, figsize=(15, 5))
    # plot a scatter plot of the residuals
    ax1.scatter(y_test, y_test - y_pred_test)
    ax1.set_xlabel('$y_{test}$')
    ax1.set_ylabel('Residual $*(y_{test} - y_{pred}) / y_{test})$')
    ax1.set_title('Residuals')
    # plot a scatter plot of the percentage error
    ax2.scatter(y_test, 100. * (y_test - y_pred_test) / y_test)
    ax2.set_xlabel('$y_{test}$')
    ax2.set_ylabel('percentage error $100*(y_{test} - y_{pred}) / y_{test})$')
    ax2.set_title('Percentage error (%)')
    # plot a histogram of the percentage error
    ax3.hist(100. * (y_test - y_pred_test) / y_test, bins=20)
    ax3.set_xlabel('percentage error $100*(y_{test} - y_{pred}) / y_{test})$')
    ax3.set_ylabel('Frequency (number of test samples)')
    ax3.set_title('Histogram of percentage error (%)')

    print(f'Test Dataset: Mean Absolute Error: {mean_absolute_error(y_test, y_pred_test):.2f}')
    print(f'Train Dataset: Mean Absolute Error: {mean_absolute_error(y_train, y_pred_train):.2f}')

    # print the equation
    print('y = ', end='')
    for i, coef in enumerate(model.coef_):
        print(f'{coef:.2f} * x_{i} + ', end='')
    print(f'{model.intercept_:.2f}')
    fig.tight_layout()

From the above we can see that, there are 10 parameters fitted into the lin_model and 65 parameters fitted into the poly_model. The 10 features that are fitted into the linear model are the originial independent variable from the loaded dataset.

However, when we transformed the X_train features into polynomial degree of 2 using Polynomial Features, 65 features got created into the transformed training set. The reason for this rise in number of features is due to the interaction terms of the independent variables.

Out of this 65 features: 10 features are original features (age, sex, bmi, bp, s1, s2, s3, s4, s5, s6)

10 features are 2nd degree terms of original features (age^2, sex^2, bmi^2, bp^2 etc..)

The remaining 45 features are interaction terms between independent variables

i.e age variable interacts with remaining 9 variables and creates 9 features (age.sex, age.bmi, age.bp, age.s1, etc...);

sex variable interacts with remaining 8 variable (age.sex interaction already considered) (sex.bmi, sex.bp, sex.s1, etc..);

bmi variable interacts with 7 variables (age.bmi, sex.bmi already considered) (bmi.bp, bmi.s1, etc);

Like this 9 + 8 + 7 + 6 + 5 + 4 + 3 + 2 + 1 = 45 interaction variables will be created.

Question-3: What insights regarding the goodness-of-fit can you draw from the graphs provided by the function below, for each of the models?¶

Answer:

lin_model Graph Analysis¶

From the graphs of muliple linear regression model (lin_model), we have the following insights regarding the goodness of fit.

In the residual plot, the residuals of lin_model for lower value y_test points are below the regression line. Whereas, the residuals with high value y_test points lies above the regression line. The residuals are concentrated more below the regression line. This makes the regression line tilted downwards slightly due to difference in residual volume.

From the percentage error plot, we can see that majority of the residual points have negative percentage error outweighing the few residuals with positive percentage error, which leads to negative percentage error. Most of the residual points swarming around the zero percentage error is an indication of some what good performance of the lin_model. Additionally, as the y_test value is increasing, the more residuals are approaching the zero residual error.

From the frequency distribution histogram of residuals for lin_model, it is evident that the residuals are normally distributed around zero. However, some of the residuals are skewed toward left side causing a negative percentage error.

poly_model Graph Analysis¶

From the graphs of the polynomial regression model (poly_model) of degree = 2, we have the following insights about goodness-of-fit.

The residual plot illustrates that the residuals are more spreadout around zero. However, just like linear model, low value y_test points have more residuals below zero and in a positive linear trend, the residuals shifted above zero value for high value y_test data. The greater residual concentration below zero results a downward tilt in the prediction line.

In the percentage error plot, the number of residuals towards the negative percentage error axis are more, leading to a negative percentage error to become more. As the y_test value is increasing, the residuals are swarming around the zero percentage error.

From the frequency distribution of residuals, we can see that, the residuals are symmetrically distributed around zero with some skewed residual distribution towards the left hand side (-ve side).

Comparison between lin_model and poly_model graphs¶

Random scatter of residuals around zero indicates a good fit, suggesting that the model captures the underlying relationships. However, if there is a clear pattern in the residuals, such as a curve, an increasing or decreasing trend, or any other systematic deviation from zero, it suggests that the model is not capturing all the information and may have room for improvement.

From our residual vs y_test graphs, we can see that, the residuals were not randomly scattered around zero rather following an increasing pattern. But lin_model has slightly better distribution compared to poly model.

Moreover, The percentage error graphs illustrate the dominance of negative percenatge error in both models. But, unlike poly_model, lin_model have more points swarmed around zero percentage error. Related to residual trend, both model following the same increasing trend.

Finally, after examining the distribution of the residuals, we can see that the residuals of both poly_model and lin_model follow a normal distribution around zero. A normal distribution of residuals represents that the model's errors are unbiased and have constant variance. However, more lin_model residuals are distributed around zero than poly_model residuals.

Hence, by comparing the graphs developed above, we can see that linear regression model goodness-of-fit characteristics are slightly better compared to polynomial regression model.
