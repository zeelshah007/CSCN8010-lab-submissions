Bar Chart¶

Control the bar color with label parameter

In [109]:
fig, ax = plt.subplots(figsize=(4,5))

fruits = ['tomatto', 'chilly', 'brinjal', 'onion']
counts = [40, 100, 30, 55]
bar_labels = ['red', 'green', 'black', 'purple']
bar_colors = ['tab:red', 'tab:blue', 'black', 'purple']

ax.bar(fruits, counts, label=bar_labels, color=bar_colors)

ax.set_ylabel('fruit supply')
ax.set_title('vegitables availability',fontsize=12,fontweight='bold')
ax.legend(title='vegitable color')

plt.show()

how to annotate a plot with an arrow pointing in a chart

fig, ax = plt.subplots(figsize=(7,3))

t = np.arange(0.0, 5.0,0.5)
s = np.cos(2*np.pi*t)
line, = ax.plot(t, s, lw=2)

ax.annotate('the local max', xy=(4, 1), xytext=(3, 1.5),
            arrowprops=dict(facecolor='red'),
            )
ax.set_ylim(-2, 2)
plt.show()
